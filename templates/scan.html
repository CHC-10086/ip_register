为什么要安装 Npcap？
  • ARP 扫描：可以获取设备的 MAC 地址和厂商信息
  • Ping 扫描：仅能发现在线 IP，无法获取 MAC 地址

两种扫描模式对比：

功能 ARP 扫描（需 Npcap） Ping 扫描（无需安装）
发现在线设备 ✓ ✓
获取 MAC 地址 ✓ ✗
识别设备厂商 ✓ ✗
需要管理员权限 是 否
安装步骤
  1. 下载 Npcap 安装包
  2. 以管理员身份运行安装程序
  3. 勾选 Install Npcap in WinPcap API-compatible Mode
  4. 完成安装后重启应用

使用说明
扫描模式

系统会自动检测 Npcap 是否可用：有 Npcap 时使用 ARP 扫描（可获取 MAC），没有时自动降级为 Ping 扫描（仅发现在线 IP）。

快速登记
  • 扫描结果中点击「登记」按钮可快速录入设备信息
  • IP、MAC、端口会自动填充
  • 填写设备名、使用者等信息后保存即可
  • 支持「批量登记」所有未录入设备
